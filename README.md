# node-docx

> markdown文当平台，支持文件夹名配置，文档排序，浏览。

## 配置参数

```
{
    // 监听端口
    "port": "8888",
    // markdown文件目录
    "path": "public/list",
    // 站点标题
    "title": "小常",
    // 站点名称/logo
    "name": "CJC-DOC",
    // 站点名称默认跳转地址
    "link":"/",
    // 搜索文案
    "search": "全文搜索",
    // 静态文件目录
    "static": "public",
    // 忽略的文件和文件夹
    "ignoredir": ["img", "imgs", "image", "images", "css", "style", "script", "js", "javascript", ".svn", ".git", ".DS_Store", ".html", ".htm", ".js", ".css"],
    // 默认展现的文件目录
    "index": "/list/README.md"
}
```
