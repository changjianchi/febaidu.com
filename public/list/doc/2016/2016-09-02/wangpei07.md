## MIP

### 1. 校验规则

#### 背景与目标

    mip-validator 是一个支持 npm 安装，同时支持网页校验的 MIP 校验工具， 校验规则就是为校验工具提供一套符合 MIP 规范的规则。

#### 完成情况

- 进度：

    - 校验规则升级

    ```
    1. MIP 相对路径检查：
        -『禁用』：以单斜线开始，如'/ext/'
        -『允许』： 1）双斜线'//', 2）'../', 3）直接字母 等

        升级细节：
            - link 的 href
            - mip-img 的 src
            - mip-video 的src 和poster
            - mip-audio

    2. input 标签校验规则升级, 允许父节点为 mip-form 的input标签
    ```
