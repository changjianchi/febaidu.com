## 1. 校验规则（无更新）

### 背景与目标

    mip-validator 是一个支持 npm 安装，同时支持网页校验的 MIP 校验工具， 校验规则就是为校验工具提供一套符合 MIP 规范的规则。

### 完成情况

- 进度：

    - 校验规则升级

    ```
    1. 不允许 html 标签中含有 style 属性

    2. 不允许 html 标签中含有 on开头的属性（如onclick），但是允许适用 on
    ```
