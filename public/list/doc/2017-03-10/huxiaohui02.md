#### 专家平台-技术优化

- **问题描述**：当打包构建应用时，会把所有js代码打包为一个文件。现在页面越来越多，会使Javascript包十分庞大，影响页面加载。

- **问题解决**：结合 Vue 的 异步组件 和 Webpack 的 code splitting feature, 实现了路由组件的懒加载。把不同路由对应的组件分割成不同的代码块，然后当路由被访问的时候才加载对应组件，更加高效。
